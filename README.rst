======
QBash
======

QBash embeds a Bash shell inside a Qt widget.

REQUIREMENTS
============

* Python 3.x
* PyQt4
* `Pyte <https://github.com/selectel/pyte>`_.

INSTALLATION
============

.. code-block:: bash

   git clone https://github.com/qtmacsdev/qbash.git
   cd qbash/bin
   python3 demo.py
